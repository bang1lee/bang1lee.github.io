# 🐍 Python Antigravity 프로젝트

Python Antigravity 패키지를 활용한 웹/앱 프로젝트 모음입니다.

## 시작하기

```bash
# 가상환경 생성
python3 -m venv venv

# 가상환경 활성화 (Mac/Linux)
source venv/bin/activate

# 가상환경 활성화 (Windows)
venv\Scripts\activate

# Antigravity 설치
pip install antigravity

# 기타 패키지 설치
pip install flask requests
```
