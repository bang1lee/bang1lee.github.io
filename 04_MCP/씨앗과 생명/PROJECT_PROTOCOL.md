## 1. 비전과 미션

**비전**: 모든 아이가 씨앗에서 생명의 가치를 발견하고, 자연과 신앙 안에서 성장하는 세상

**미션**: 바이오필리아 교육, 성경 프로그램, 다큐멘터리 미디어를 통해 아이들(6-12세)이 씨앗, 기후, 생명, 성경의 연결을 체험하는 비영리 교육 플랫폼을 운영합니다.

**핵심 키워드**: 바이오필리아(Biophilia) — E.O. Wilson의 자연에 대한 본능적 유대, 창조 청지기직(Creation Stewardship) — 성경적 환경 돌봄의 사명, 토종 씨앗 보존 — 기후 위기 대응과 생물다양성 보전, 미디어 리터러시 — 다큐멘터리 제작을 통한 관찰력과 표현력 양성

## 4. 워크플로우 프로토콜

### 4-1. 리서치 → 목업 리뉴얼 반복 사이클

```
┌─────────────────────────────────────────────────┐
│  CYCLE: Research → Mockup Renewal               │
│                                                 │
│  ① Audit     현재 상태 분석 (코드/디자인/성능)      │
│      ↓                                          │
│  ② Research  경쟁사/트렌드/기술 리서치              │
│      ↓                                          │

│      ↓                                          │
│  ④ Build     React 목업 제작/업데이트               │
│      ↓                                          │
│  ⑤ Review    브라우저에서 검증 + 피드백              │
│      ↓                                          │

└─────────────────────────────────────────────────┘
```

### 4-2. Claude 사용법

새 사이클 시작: "포트폴리오 리뉴얼 사이클 돌려줘" / "리서치하고 목업 업데이트해줘"

특정 단계만: "경쟁사 리서치만 해줘" (②), "현재 목업 성능 체크해줘" (①), "리서치 결과 반영해서 목업 수정해줘" (④)

특정 영역 집중: "모바일 반응형 개선 사이클 돌려줘" / "SEO 최적화 리서치하고 반영해줘"

### 4-3. 각 단계 상세

**① Audit**: Lighthouse 점수 체크, 코드 품질 분석, 반응형 테스트 (320px~2560px), 이전 사이클 대비 비교

**② Research**: Awwwards/Behance/Dribbble 트렌드, 최신 React 패턴, UX/접근성 가이드라인, Core Web Vitals 최적화

**③ Plan**: 리서치 기반 개선 항목 도출, Critical → Major → Minor 우선순위, 작업 범위 산정

**④ Build**: JSX/HTML 업데이트, 새 컴포넌트/섹션 추가, 디자인 시스템 일관성 유지

**⑤ Review**: 브라우저 렌더링 확인, 인터랙션 테스트, 크로스 브라우저 체크

**⑥ Iterate**: 피드백 수집 반영, 버전 히스토리 관리, 다음 사이클 계획

## 8. 기술 아키텍처

### 8-1. 프론트엔드 스택

| 기술 | 용도 |
|---|---|
| HTML5 (시맨틱) | 구조 — nav, main, section, footer, article |
| CSS3 Custom Properties | 디자인 토큰 — 색상, 타이포, 간격, 효과 |
| CSS Grid + Flexbox | 레이아웃 — 반응형 그리드 시스템 |
| CSS clamp() | 유체 타이포그래피 — 뷰포트 기반 자동 크기 |
| Vanilla JavaScript | 인터랙션 — 의존성 제로 |
| IntersectionObserver | 스크롤 리빌 + 카운터 + lazy loading |
| CSS @media queries | 반응형 — 480px, 768px, 1024px 브레이크포인트 |

### 8-2. 디자인 시스템

**컬러 팔레트**: Green (900~50) — 주요 색상, Gold (600~300) — 강조, Cream — 배경, Sky (700~300) — 텔레그램 섹션, Brown (600~300) — 보조

**타이포그래피**: Playfair Display (디스플레이/제목) + Noto Sans KR (본문/UI)

**간격 시스템**: 4px 기반 (space-1: 0.25rem ~ space-32: 8rem)

**모서리 반경**: sm(0.375rem), md(0.75rem), lg(1rem), xl(1.5rem), 2xl(2rem)

**그림자**: sm, md, lg, xl — rgba(26, 58, 42) 기반

**전환**: fast(0.2s), base(0.35s), slow(0.5s) — cubic-bezier(0.16, 1, 0.3, 1)

### 8-3. 접근성 (WCAG 2.1 AA)

## 15. 바이오필리아 교육 리서치 요약

**핵심 연구**: E.O. Wilson의 바이오필리아는 인간이 생명과 자연에 대해 갖는 본능적 유대감으로, 2세 이전부터 관찰되며 11세 이전 자연 탐험 경험이 평생의 환경 가치관을 형성함

**발달 단계별 설계**: 공감(4-7세, 자연과 감정적 연결) → 탐구(7-11세, 주변 세계 이해) → 사회적 행동(11세+, 환경 보호 실천)

**설계 원칙**: 비구조적 야외 환경이 가장 효과적, 매일 확장된 야외 시간 필요, 어른의 자연 존중 모델링 중요, "야생에 가까운" 환경일수록 효과적

**바이오필릭 디자인**: 자연 패턴/이미지 → 기억력·주의력 향상, 자연 채광/식물 → 저비용 고효과, 교사·학생 웰빙 향상

**국제 사례**: 독일 숲유치원 700+개, 스칸디나비아·스위스 확산, 한국 유아 숲체험원 도입 중
