## 1. 계정 개설

### 비즈니스 계정 설정

1. **계정 생성**: `@seeds_and_life`
2. **비즈니스 계정 전환**: 설정 → 계정 → 프로페셔널 계정으로 전환
3. **카테고리 선택**: `비영리 단체` 또는 `교육`
4. **연락처 설정**: 이메일, 전화번호, 주소 추가
5. **Instagram Insights 활성화** (비즈니스 계정 자동)

### 비영리 배지 신청

Meta에서 제공하는 비영리 기능:
- **기부 버튼** 추가 가능 (Instagram Fundraising)
- **비영리 인증 배지** 신청 (Giving Tools)
- 스토리/라이브에서 **기부 스티커** 사용 가능

신청 방법:
1. [Meta for Nonprofits](https://nonprofits.fb.com/) 등록
2. 비영리 단체 인증서류 제출
3. 승인 후 기부 도구 활성화

## 3. 콘텐츠 전략

### 콘텐츠 믹스 (4:3:2:1 비율)

| 유형 | 비율 | 예시 |
|---|---|---|
| **교육/가치** | 40% | 씨앗 지식, 성경 식물, 바이오필리아 팁 |
| **활동/현장** | 30% | 수업 현장, 아이들 활동, 비하인드 |
| **스토리/감성** | 20% | 아이들 인터뷰, 성장 이야기, 자연 사진 |
| **참여/CTA** | 10% | 신청 안내, 후원 요청, 이벤트 |

### 포스트 유형별 전략

#### 피드 포스트 (주 3-5회)
- **캐러셀**: 교육 콘텐츠 (성경 식물 5가지, 씨앗 보존 팁 등)
- **단일 이미지**: 현장 사진, 자연 관찰, 활동 사진
- **인용구**: 성경 구절 + 자연 이미지 조합

#### 릴스 (주 1-3회)
- **30초 교육**: "오늘의 씨앗 이야기", "성경 속 이 식물은?"
- **타임랩스**: 씨앗 발아 과정, 계절 변화
- **비하인드**: 수업 준비, 교재 만들기 과정
- **아이들 반응**: (보호자 동의 후) 아이들의 "오늘 배운 것"

#### 스토리 (매일)
- **일상 공유**: 오늘의 교실, 날씨와 자연
- **투표/질문**: "어떤 씨앗이 궁금하세요?", "좋아하는 성경 식물은?"
- **카운트다운**: 프로그램 시작일 D-day
- **실시간**: 수업 현장 라이브 공유

## 4. 비영리 특화 전략

### 핵심 원칙 (2025 비영리 SNS 트렌드 반영)

1. **진정성 > 완벽함**: 전문 촬영보다 진솔한 현장 사진이 더 높은 참여를 이끌어냄
2. **사람 중심**: 얼굴이 보이는 포스트가 참여율 2-3배 높음
3. **연결 > 팔로워**: 소수의 진심 어린 커뮤니티가 다수의 소극적 팔로워보다 가치 있음
4. **가치 제공**: 팔로워에게 실질적인 교육 가치를 제공하는 콘텐츠

### 해시태그 전략

```
브랜드 태그 (항상 포함):
#씨앗과생명 #SeedsAndLife #씨앗교육

주제별 태그 (3-5개 선택):
교육: #바이오필리아 #자연교육 #어린이교육 #비영리교육 #환경교육
신앙: #성경교육 #어린이신앙 #교회학교 #주일학교 #신앙교육
자연: #토종씨앗 #기후교육 #생태교육 #자연관찰 #숲교육
미디어: #다큐멘터리 #미디어교육 #영상교육

커뮤니티 태그 (1-2개):
#비영리 #사회적가치 #교육혁신 #지속가능교육
```

### 커뮤니티 빌딩

- **DM 자동응답**: 새 팔로워에게 환영 메시지 + 프로그램 안내
- **댓글 응답**: 모든 댓글에 24시간 이내 응답
- **UGC 활용**: 학부모가 공유한 사진 리그램 (동의 후)
- **콜라보**: 교회, 환경단체, 교육 기관 계정과 협업 포스트

## 5. 성장 로드맵

### Phase 1: 론칭 (1-2개월)

- [ ] 계정 생성 및 프로필 최적화
- [ ] 하이라이트 10개 이상 제작
- [ ] 피드 포스트 15개 이상 미리 준비 후 론칭
- [ ] 관련 계정 100개 이상 팔로우 및 소통
- [ ] 교회, 교육기관 네트워크 알림

**목표**: 팔로워 200명, 주 3회 피드 포스트

### Phase 2: 성장 (3-6개월)

- [ ] 릴스 주 2회 이상 발행
- [ ] 스토리 매일 1회 이상
- [ ] 교회/기관 콜라보 월 1회
- [ ] 인스타 라이브 월 1회 (프로그램 소개)
- [ ] 유료 광고 테스트 (월 5만원)

**목표**: 팔로워 1,000명, 평균 참여율 5%+

### Phase 3: 확장 (7-12개월)

- [ ] 인플루언서 협업 (교육/육아 분야)
- [ ] 릴스 바이럴 전략 (씨앗 타임랩스 시리즈)
- [ ] 인스타 숍 (교육 키트 판매 검토)
- [ ] 크로스 플랫폼 (유튜브, 틱톡 연동)

**목표**: 팔로워 5,000명, 프로그램 신청 전환율 측정
