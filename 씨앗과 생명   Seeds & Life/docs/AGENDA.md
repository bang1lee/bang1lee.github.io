# 🌱 씨앗과 생명 — 거대 아젠더 (Grand Agenda)

> 내부 커뮤니케이션과 외부 커뮤니케이션을 통합한 비영리 교육 플랫폼 운영 전략

---

## 1. 비전과 미션

### 비전
**모든 아이가 씨앗에서 생명의 가치를 발견하고, 자연과 신앙 안에서 성장하는 세상**

### 미션
바이오필리아 교육, 성경 프로그램, 다큐멘터리 미디어를 통해 아이들(6-12세)이 씨앗, 기후, 생명, 성경의 연결을 체험하는 비영리 교육 플랫폼을 운영합니다.

### 핵심 키워드
- **바이오필리아(Biophilia)**: 인간의 자연에 대한 본능적 유대 (E.O. Wilson)
- **창조 청지기직(Creation Stewardship)**: 성경적 환경 돌봄의 사명
- **토종 씨앗 보존**: 기후 위기 대응과 생물다양성 보전
- **미디어 리터러시**: 다큐멘터리 제작을 통한 관찰력과 표현력 양성

## 2. 이해관계자 맵

### 외부 이해관계자 (External Stakeholders)

| 이해관계자 | 니즈 | 소통 채널 | 핵심 메시지 |
|---|---|---|---|
| **학부모** | 아이의 전인적 성장, 안전, 가치 교육 | 홈페이지, 텔레그램 봇, 인스타그램 | "아이가 자연과 신앙 안에서 스스로 자라는 경험" |
| **아이들 (6-12세)** | 재미, 탐험, 친구, 발견 | 프로그램 내 직접 경험 | "씨앗 하나가 얼마나 신기한지 알아?" |
| **교회/주일학교** | 신앙교육 콘텐츠, 주일학교 프로그램 | 홈페이지, 파트너십 제안 | "성경과 자연이 만나는 커리큘럼" |
| **지역 농가** | 토종 씨앗 보존 동참, 교육 체험 장소 제공 | 직접 소통, 파트너십 | "토종 씨앗의 가치를 다음 세대에 전합니다" |
| **환경/교육단체** | 협업, 자원 공유, 네트워크 확장 | 공식 제안서, 미디어 협업 | "바이오필리아 교육의 새로운 모델" |
| **후원자/기업 CSR** | 사회적 임팩트, 투명한 운영 | 뉴스레터, 인스타그램, 보고서 | "측정 가능한 교육적 임팩트" |

### 내부 이해관계자 (Internal Stakeholders)

| 이해관계자 | 역할 | 소통 채널 | 필요 도구 |
|---|---|---|---|
| **운영 총괄** | 전체 전략, 파트너십, 재정 | 운영진 텔레그램 채널 | 대시보드, KPI 트래킹 |
| **교육팀** | 커리큘럼 개발, 수업 진행 | 교육팀 텔레그램 채널 | 교안 아카이브, 일정 관리 |
| **미디어팀** | 다큐멘터리 제작, 사진/영상 | 미디어팀 텔레그램 채널 | 촬영 일정, 에셋 관리 |
| **봉사자** | 현장 지원, 아이들 케어 | 봉사자 텔레그램 채널 | 교육 자료, 일정 |

## 3. 커뮤니케이션 아키텍처

### 3-1. 외부 커뮤니케이션 (신청 고객 대상)

```
학부모 여정 (Customer Journey):

인지 → 관심 → 신청 → 참여 → 유지/추천

[인지]
├── 인스타그램 콘텐츠 발견
├── 교회/학교 입소문
├── 검색 (SEO)
└── 타 기관 추천

[관심]
├── 홈페이지 방문 → 프로그램 탐색
├── 인스타그램 팔로우 → 릴스/스토리 시청
└── 텔레그램 봇 → 프로그램 문의

[신청]
├── 홈페이지 신청 폼 작성
├── 텔레그램 봇 → 자동 접수 확인
└── 운영진 → 수동 확인 및 안내

[참여]
├── 텔레그램 커뮤니티 그룹 초대
├── 프로그램 일정/준비물 안내
├── 활동 사진 텔레그램 공유
└── 인스타그램 스토리 공유

[유지/추천]
├── 수료 후 다음 프로그램 안내
├── 커뮤니티 지속 참여
├── 후원 전환 유도
└── 입소문 (UGC 콘텐츠 제작)
```

### 3-2. 내부 커뮤니케이션 (운영진 대상)

```
운영 흐름:

[신청 접수]
├── 홈페이지 폼 → 텔레그램 봇 → 운영진 알림
├── Google Sheets 자동 기록 (n8n 연동)
└── 접수 확인 메시지 자동 발송

[프로그램 운영]
├── 주간 회의 (텔레그램 운영진 채널)
├── 커리큘럼 교안 공유 (리소스 아카이브)
├── 촬영/미디어 일정 조율
└── 봉사자 배정 및 교육

[성과 관리]
├── KPI 대시보드 (주간/월간)
│   ├── 신청자 수 / 전환율
│   ├── 참여율 / 만족도
│   ├── SNS 성장 지표
│   └── 재정 현황
└── 월간 리포트 작성 및 공유
```

## 6. 비영리 지속가능성 전략

### 수익 구조 (Revenue Mix)

```
목표 연간 예산: 50,000,000원

수익원 구성:
├── 기업 CSR 파트너십     35%  (17,500,000원)
├── 재단/기관 보조금       25%  (12,500,000원)
├── 프로그램 참가비         15%  ( 7,500,000원)
├── 미디어 콘텐츠 라이선스  10%  ( 5,000,000원)
├── 커뮤니티 후원           10%  ( 5,000,000원)
└── 기타 (굿즈, 워크숍)     5%  ( 2,500,000원)
```

### 핵심 전략

#### 1. 교육 프로그램 수익화
- **기본 프로그램**: 무료 (접근성 확보)
- **심화 캠프**: 최소 참가비 (3-5만원)
- **기업 CSR 연계**: 삼성, SK, 현대 등 ESG 프로그램 파트너십
- **교사 연수**: 유료 전문 워크숍 (바이오필리아 교수법)

#### 2. 미디어 콘텐츠 라이선스
- **교회/주일학교**: 커리큘럼 영상 라이선스 (월 구독)
- **학교/기관**: 다큐멘터리 상영권
- **유튜브**: 교육 채널 광고 수익 (장기)
- **출판**: 교재/워크북 출판

#### 3. 파트너십 네트워크
- **교회 네트워크**: 전국 교회 주일학교 프로그램 보급
- **지역 농가**: 교육 체험 장소 + 토종 씨앗 제공
- **환경단체**: 공동 프로그램 운영 (국립생태원, 수목원 등)
- **대학**: 교육학/생태학 연구 협력

#### 4. 커뮤니티 후원
- **정기 후원**: 월 정기 후원자 모집 (1만원~)
- **씨앗 후원**: 토종 씨앗 1봉 = 아이 1명 교육 후원
- **재능 봉사**: 미디어 제작, 교육 보조, 농사 지도
- **크라우드펀딩**: 계절 캠프 특별 프로젝트

### 임팩트 측정 (Impact Metrics)

| 지표 | 측정 방법 | 목표 (연간) |
|---|---|---|
| 교육 참여 아동 수 | 프로그램 등록/출석 | 200명 |
| 커리큘럼 완수율 | 출석 + 프로젝트 완성 | 85% |
| 보호자 만족도 | 설문조사 (10점 척도) | 9.0+ |
| 자연 감수성 변화 | 사전/사후 평가 | +30% |
| 미디어 콘텐츠 도달 | 조회수 + 라이선스 수 | 50,000회 |
| 토종 씨앗 보존 종 | 아카이브 등록 | 50종 |
| 파트너 기관 수 | MOU/협약 | 20곳 |
